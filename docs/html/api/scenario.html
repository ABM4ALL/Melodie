Scenario

class Melodie.scenario_manager.Scenario(id_scenario: Optional[Union[int, str]] = None)

Bases: Melodie.element.Element

Scenario contains a set of parameters used in simulation model. It is created before the initialization of Model.

Parameters

id_scenario – the id of scenario. if None, this will be self-increment from 0 to scenarios_number-1
