rand_move_agent(agent, category, range_x, range_y)

Randomly move an agent with maximum movement range_x in x axis and range_y in y axis.

Parameters:

  • agent Must be Melodie.GridAgent, not Agent. That is because GridAgent has predefined properties required in Grid.

  • range_x The activity range of agent on the x axis.

  • range_y The activity range of agent on the y axis.

For example, if the agent is at (0, 0), range_x=1 and range_y=0, the result can be (-1, 0), (0, 0) or (1, 0). The probability of these three outcomes are equal.

Returns:

(int, int), the new position

setup_agent_locations(category, initial_placement='direct')

Add an agent category.

For example, if there are two classes of agents: Wolf(GridAgent) and Sheep(GridAgent), and there are 100 agents with id 0~99 for each class. It is obvious in such a circumstance that we cannot identify an agent only with agent id.So it is essential to use category_name to distinguish two types of agents.

Parameters:

  • category_id The id of new category.

  • category An AgentList object

  • initial_placement A str object stand for initial placement.

Return type:

None

Returns:

None

setup_params(width, height, wrap=True, caching=True, multi=True)

Setup the parameters of grid.

Parameters:

  • width (int) int

  • height (int) int

  • wrap bool, True by default.

If True, GridAgent will re-enter the grid on the other side if it moves out of the grid on one side. :type caching: :param caching: bool, True by default. If true, the grid caches the neighbor of each spot. :type multi: :param multi: bool, True by default. If true, more than one agent could stand on one spot. If false, error will be raised when attempting to place multiple agents on one spot. :return: None

to_2d_array(attr_name)

Collect property of each spot and write the property value into an 2d np.array.

Notice:

  • The property to collect should be float/int/bool, not other types such as str.

  • If you would like to get an element from the returned array, please write like this:

1arr = self.to_2d_array('some_attr')
2y = 10
3x = 5
4spot_at_x_5_y_10 = arr[y][x] # CORRECT. Get the some_attr value of spot at `x = 5, y = 10`
5spot_at_x_5_y_10 = arr[x][y] # INCORRECT. You will get the value of spot at `x = 10, y = 5`
Parameters:

attr_name (str) the property name to collect for this model.

Return type:

ndarray

Returns:
