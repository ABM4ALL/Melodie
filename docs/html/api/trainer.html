class Melodie.Trainer(config: Config, scenario_cls: Type[Scenario] | None, model_cls: Type[Model] | None, data_loader_cls: Type[DataLoader] | None = None, processors: int = 1)ÔÉÅ

Bases: BaseModellingManager

Trainer trains the agents to update their behavioral parameters for higher payoff.

Parameters:

  • config ‚Äì Config instance for current project.

  • scenario_cls ‚Äì Scenario class for current project.

  • model_cls ‚Äì Model class in current project.

  • data_loader_cls ‚Äì DataLoader class in current project.

  • processors ‚Äì Each path in current iteration will be computed parallelly, this parameter stands for processor cores used in parallel computation.
